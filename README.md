# Smart G Networks - Central Stock & Branch Transfer System

A comprehensive inventory management system for managing multiple branches, stock transfers, sales tracking, and financial reporting.

## Features

- **Multi-branch Management**: Admin can create/manage branches and branch managers
- **Central Stock Control**: Admin manages central inventory and transfers to branches
- **Stock Transfer Approvals**: Branches approve/receive stock transfers
- **Point of Sale**: Branches record sales with automatic stock deduction
- **Sales History**: Track all sales with commission calculations
- **Inventory Management**: Real-time stock levels with low stock alerts
- **Financial Reports**: Company profit/loss statements, branch reports
- **Expense Tracking**: Record branch expenses and shortages
- **Export Capabilities**: CSV and PDF export for all reports

## Tech Stack

- **Backend**: Python Flask
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Authentication**: JWT
- **Password Hashing**: bcrypt
